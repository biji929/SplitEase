# SplitEase

SplitEase is an Android app designed to make splitting bills with friends as simple as possible. Whether you're sharing the cost of dinner, a road trip, or monthly rent, the app lets you add a bill, choose who's involved, and automatically calculates how much each person owes.

## Features

When you open the app, you're greeted with an onboarding splash screen before being taken to the login page. From there you can sign in to an existing account or create a new one through the register screen. Once logged in, the home dashboard gives you a quick overview of your recent bills and current balances. You can add a new bill at any time by giving it a title, entering the total amount, selecting a category, and choosing which friends are splitting it. The app then divides the cost equally and shows you a summary screen with the payment status of each person. If someone hasn't paid yet, you can send them a reminder directly from the app. There's also a logout confirmation so you don't accidentally sign out.

## Tech Stack

The app is built with Kotlin using XML layouts and Material 3 components. It targets Android SDK 34 and supports devices running Android 7.0 and above.

## How to Run

Clone the repository, open it in Android Studio, sync the Gradle files, and press the run button. You can log in using the default demo credentials — name is `admin` and password is `1234`.
