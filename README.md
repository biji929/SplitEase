💸 SplitEase — Bill Splitting App
A modern Android app that makes splitting bills with friends easy and hassle-free.

📱 Features

Splash Screen — Onboarding screen with app branding
Login Screen — Secure sign-in with name and password
Register Screen — Create a new account
Home Dashboard — View recent bills and balances
Bill Splitting — Split bills equally among friends
Payment Status — Track who has paid and who hasn't
Send Reminders — Notify unpaid members
